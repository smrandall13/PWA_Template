Page Example 1

This information serves to inform you on how to edit this app and add your own pages.


app.json

The app.json file is used to configure the app. You can change the App Name, Icon, version, and available Pages.


The pages is an array of objects with the following properties:
- id: The id of the page
- name: The name of the page
- icon: The icon of the page
- html: The html of the page
- style: The style of the page
- script: The script of the page

Note that the script and style properties are optional. 



{
	"name": "PWA Template", "icon": "/icons/app-icon.png",
	"version": "1.0.0",
	"pages": [
		{
			"id": "example01",
			"name": "Example 01", -- This Page
			"icon": "/pages/cross-white.png",
			"html": "/pages/example01.html",
			"style": "/pages/example01.css",
			"script": "/pages/example01.js"
		},
		{
			"id": "example02",
			"name": "Example 02",
			"html": "/pages/example02.html",
			"style": "/pages/example02.css",
			"script": "/pages/example02.js"
		}
	]
}

If you would like your page assets to be available offline you can add them to the ASSETS_CUSTOM array in the service-worker.js file.
